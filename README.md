## 🚀 Quick Start

### Installation

1. **Download** the latest release from [GitHub Releases](https://github.com/indfir/SmartTools-ConverterAllinOne/releases)
2. **Extract** the ZIP file to any location
3. **Double-click** `SmartTools.exe` to launch

> ✅ No installation required! The application runs immediately without any setup.

### Requirements

- **Windows 7** or newer
- No additional software installation needed
- Internet connection required (for CDN-based libraries)

## 🛠️ Technology Stack

### Core Framework
- **[NW.js](https://nwjs.io/)** - Chromium-based desktop runtime for cross-platform apps

### PDF Processing
- **[PDF-Lib](https://pdf-lib.js.org/)** - PDF creation and manipulation
- **[PDF.js](https://mozilla.github.io/pdf.js/)** - Mozilla's PDF rendering engine

### Document Handling
- **[XLSX](https://github.com/SheetJS/sheetjs)** - Excel file parsing and generation
- **[Mammoth](https://github.com/mwilliamson/mammoth.js)** - DOCX document conversion
- **[PptxGenJS](https://gitbrent.github.io/PptxGenJS/)** - PowerPoint generation
- **[docx](https://github.com/dolanmiu/docx)** - DOCX document creation

### Image & Export
- **[html2pdf.js](https://github.com/eKoopmans/html2pdf.js)** - HTML to PDF conversion
- **[QRCode](https://github.com/davidshimjs/qrcode)** - QR code generation
- **[jsQR](https://github.com/cozmo/jsqr)** - QR code reading
- **[JSZip](https://stuk.github.io/jszip/)** - ZIP file handling

### UI & Rendering
- **[Phosphor Icons](https://phosphoricons.com/)** - Beautiful, consistent icon set
- **[Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans)** - Modern typeface

## 📁 File Structure

```
SmartTools/
│
├── SmartTools.exe          # Main executable (double-click to run)
├── index.html              # Application HTML interface
├── styles.css              # Application styling
├── app.js                  # Core application logic
├── icon.png                # Application icon
├── package.json            # NW.js configuration & dependencies
├── package-lock.json       # Dependency lock file
├── README.md               # This documentation

├── lib/                    # JavaScript libraries
│   ├── docx.umd.js         # DOCX document handling
│   ├── html2pdf.bundle.min.js  # HTML to PDF conversion
│   ├── jsQR.min.js         # QR code reading

│   ├── mammoth.browser.min.js  # DOCX parsing
│   ├── marked.min.js       # Markdown rendering
│   ├── pdf-lib.min.js      # PDF manipulation
│   ├── pdf.min.js          # PDF.js main library
│   ├── pdf.worker.min.js   # PDF.js worker
│   ├── pptxgen.bundle.js   # PowerPoint generation
│   ├── qrcode.min.js       # QR code generation
│   └── xlsx.full.min.js    # Excel file handling
│
├── locales/                # Internationalization files
├── swiftshader/            # GPU acceleration libraries
│
└── [NW.js Runtime Files]   # Chromium & Node.js runtime
    ├── nw.exe              # NW.js executable
    ├── nw.dll              # NW.js core library
    ├── libEGL.dll          # Graphics library
    ├── libGLESv2.dll       # OpenGL ES library
    └── ...
```

> ⚠️ **Important:** Do not delete the `nwjs` runtime files. The application requires them to run.

## 🔧 Troubleshooting

### App won't open?

1. **Verify files** - Ensure all files are in the folder (especially `nwjs` folder)
2. **Check antivirus** - Some antivirus software may flag new `.exe` files as false positives
3. **Windows version** - Requires Windows 7 or newer

### Conversion fails?

1. **File format** - Ensure the uploaded file matches the expected format
2. **File corruption** - Try with a different file to rule out corruption
3. **Internet connection** - Some libraries load from CDN on first use

### File won't save?

1. **Check permissions** - Ensure you have write access to the download folder
2. **Disk space** - Verify sufficient disk space is available
3. **Use "Save As"** - Try selecting a different save location
